Homework CFD:SPH

2021年，哈尔滨工程大学《高等水动力学》课程作业：“基于SPH方法的舰船波浪砰击模拟”。

开始

软件依赖

  • Git
  • Rust
  • mdbook

获取仓库

git clone https://github.com/zoziha/SPH-homework.git
cd SPH-homework

使用mdbook构建文档

mdBook是一个从Markdown文件创建现代在线书籍的实用程序。
你可以通过提供的book.toml文件来构建《高等水动力学作业》。

cd doc && mdbook build
